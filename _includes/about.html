Ingress is an augmented reality game created by Niantic Labs, a division of Google,in this game two teams, the Resistance and the Enlightened. In the game, the Enligtened seek to capture territory for the shapers, alien beings from another dimension. Meanwhile, the Resistance seeks defend humanity from this threat using the same technology. This is done by visiting locations known as portals, caputing them, and using them to claim more area by creating fields.

Ingress can be best described as a highly competitive geocaching game with elements of Capture the Flag and Risk added. The game is as easy as walking down the street, or can be as complex as organizing an army. It all depends on you, and how far you are willing to go.
